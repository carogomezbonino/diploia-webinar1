## 🚀 Despliegue Súper Fácil en GitHub

### Pasos:

1. **Crear repositorio en GitHub**
   - Nombre: `webinar-carina-lion`
   - Público ✅

2. **Subir estos archivos**
   - Arrastra todos los archivos a GitHub
   - O usa: `git add .` → `git commit -m "Add webinar site"` → `git push`

3. **Activar GitHub Pages**
   - Settings > Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
   - Save

### 🌐 Tu sitio estará en:

